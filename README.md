### Option 1: Quick Deploy (Easiest)

1. **Create a new repository on GitHub**
   - Go to https://github.com/new
   - Name it: `christmas-cracker` (or whatever you like)
   - Make it **Public**
   - Click "Create repository"

2. **Upload your file**
   - Click "uploading an existing file"
   - Drag and drop `index.html` 
   - Click "Commit changes"

3. **Enable GitHub Pages**
   - Go to repository Settings
   - Scroll down to "Pages" (in left sidebar)
   - Under "Source", select "main" branch
   - Click "Save"

4. **Done!** ðŸŽ‰
   - Your site will be live at: `https://yourusername.github.io/christmas-cracker`
   - It may take 1-2 minutes to deploy

### Option 2: Using Git (For developers)
